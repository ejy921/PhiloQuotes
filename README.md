## Characters (endpoint parameters)
A characteristic field was added to the quotes that contains 3 comma-seperated 
words that assign the quote values from a list of 14 total characters:

These are:
1. Analytical
2. Inquisitive
3. Modern
4. Pragmatic
5. Empathetic
6. Skeptical
7. Introspective
8. Philosophical
9. Visionary

11. Spiritual
